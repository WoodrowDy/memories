# memories

개발 지식과 개인 학습을 쌓는 **위키 + TIL** 저장소.
`daily`에 빠르게 기록하고, 익은 것을 `topics`로 정제해 위키로 굳힌다.

> 처음이라면 → **[docs/start-here.md](docs/start-here.md)** (할 일 · 매뉴얼 · 방향성 한 장 요약)

## 지도

- [daily/](daily/) — TIL 스트림 (날것 기록)
- [topics/](topics/) — 정제된 지식 위키 (evergreen)
  - [os](topics/os/) · [db](topics/db/) · [lang](topics/lang/) · [infra](topics/infra/) · [cs](topics/cs/) · [ai](topics/ai/) · [tools](topics/tools/)
- [projects/](projects/) — 끝이 있는 목표 (진행/완료)
- [personal/](personal/) — 개인 영역 (영어·스페인어 등)

## 3줄 사용법

1. 매일 배운 건 `daily/<연도>/<YYYY-MM-DD>.md`에 빠르게.
2. 건질 내용은 `topics/<주제>/`에 **한 개념 = 한 파일**로 정제하고, daily에서 링크.
3. 폴더는 통(bucket)일 뿐, 진짜 분류는 프론트매터 `tags`와 문서 간 링크가 한다.
